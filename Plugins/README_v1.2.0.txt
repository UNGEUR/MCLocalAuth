⚡ INSTALLATION RAPIDE
════════════════════════════════════════════════════════════════

1. Arrêter le serveur Minecraft

2. Dans le dossier plugins/ du serveur :
   - Supprimer : MCLocalAuth-Spigot-v1.1.0.jar
   - Copier    : MCLocalAuth-Spigot-v1.2.0.jar

3. Redémarrer le serveur

4. Sur Discord, taper /mctest pour vérifier

✅ Aucun changement de config.yml nécessaire !
