MCLocalAuth v1.3.0 - CHANGELOG
Type: Mise a jour majeure

NOUVEAUTES
----------

1. Commande Discord renommee
   - AVANT: /auth <code>
   - MAINTENANT: /mclogin <code>
   - Tous les messages et references mis a jour

2. Commande /unlink (Admin uniquement)
   - Usage: /unlink <discord_id>
   - Reserve a l'admin configure dans config.yml (admin.discord_id)
   - Journalise toutes les tentatives
   - Note: Fonctionnalite de base implementee, liaison persistante arrive prochainement

CORRECTIONS
-----------

1. Encodage UTF-8 corrige (in-game)
   Probleme resolu: Messages in-game avec caracteres bizarres
   
   Messages corriges:
   - "reinitialise" au lieu de "reinitialisé"
   - "ajoute" au lieu de "ajoute"
   - "supprime" au lieu de "supprime"
   - "defini" au lieu de "defini"
   - "autorisees" au lieu de "autorisees"

2. Support IPv6
   - IPv4 ET IPv6 acceptes pour les IP autorisees
   - Validation avec InetAddress.getByName()
   - Commandes affectees: /auth addip, /auth setip

3. IP actuelle affichee dans kick
   - Le message de kick affiche maintenant l'IP du joueur
   - Variable %current_ip% correctement remplacee
   - Aide les joueurs a identifier leur probleme de connexion

MODIFICATIONS TECHNIQUES
-------------------------

Fichiers modifies:
- AuthAdminCommand.java: Encodage ASCII + validation IPv6
- LoginListener.java: Affichage IP actuelle + encodage
- DiscordBot.java: Renommage /auth -> /mclogin
- DiscordCommandListener.java: Handler /mclogin + /unlink admin
- config.yml: Messages sans accents + /mclogin
- plugin.yml: Version 1.3.0
- pom.xml: Version 1.3.0

MIGRATION DEPUIS v1.2.x
------------------------

Etapes:
1. Arreter le serveur
2. Remplacer le JAR par MCLocalAuth-Spigot-v1.3.0.jar
3. Configurer admin dans config.yml:
   admin:
     discord_id: "TON_DISCORD_ID"
     minecraft_uuid: ""
4. Redemarrer le serveur
5. Informer les joueurs: commande est maintenant /mclogin <code>

Note: L'ancienne configuration reste compatible
