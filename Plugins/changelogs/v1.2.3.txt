    → Messages : "Demarrage", "connecte", etc.

🎮 UTILISATION
════════════════════════════════════════════════════════════════

Configuration admin (optionnel) :
1. Ouvrir plugins/MCLocalAuth/config.yml
2. Trouver la section "admin"
3. Remplir :
   - discord_id : Ton ID Discord (clic droit > Copier l'ID)
   - minecraft_uuid : (optionnel) Ton UUID Minecraft

Pour obtenir ton UUID :
→ https://mcuuid.net/?q=TON_PSEUDO
